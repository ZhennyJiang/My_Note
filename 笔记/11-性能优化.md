## 1. 预加载子应用

在 [src/qiankun/registerMicroApp.ts](vscode-file://vscode-app/d:/Microsoft VS Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 中添加预加载策略：

```typescript
import { registerMicroApps, start, prefetchApps } from 'qiankun';

// ...existing code...

// 注册子应用后，添加预加载
export function initQiankun() {
  registerMicroApps(apps);
  
  // 预加载子应用，在浏览器空闲时提前加载
  prefetchApps(apps);
  
  start({
    prefetch: 'all', // 预加载所有子应用的静态资源
    sandbox: {
      experimentalStyleIsolation: true, // 样式隔离
    },
  });
}
```

**原因**：预加载可以在用户访问子应用前就下载好资源，显著减少子应用首次加载时间。

## 4. 应用间通信优化

在 [src/store/use-mainapp-store.ts](vscode-file://vscode-app/d:/Microsoft VS Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 中优化状态共享：

```typescript
import { initGlobalState, MicroAppStateActions } from 'qiankun';

// 定义初始状态
const initialState = {
  user: null,
  token: '',
  permissions: [],
};

// 初始化全局状态
const actions: MicroAppStateActions = initGlobalState(initialState);

// 使用防抖避免频繁更新
let updateTimer: number | null = null;
export const setGlobalState = (state: Partial<typeof initialState>) => {
  if (updateTimer) clearTimeout(updateTimer);
  updateTimer = window.setTimeout(() => {
    actions.setGlobalState(state);
  }, 16); // 约一帧的时间
};

export default actions;
```

**原因**：防抖处理避免频繁的状态更新导致性能问题。

## 7. 请求优化

在 [src/request/axios.config.ts](vscode-file://vscode-app/d:/Microsoft VS Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 中添加缓存和取消重复请求：

```typescript
import axios, { AxiosRequestConfig } from 'axios';

// 请求缓存
const cache = new Map<string, { data: any; timestamp: number }>();
const CACHE_DURATION = 5 * 60 * 1000; // 5分钟缓存

// 取消重复请求
const pendingRequests = new Map<string, AbortController>();

const generateCacheKey = (config: AxiosRequestConfig) => {
  return `${config.method}-${config.url}-${JSON.stringify(config.params)}`;
};

// ...existing code...

instance.interceptors.request.use(config => {
  const cacheKey = generateCacheKey(config);
  
  // 取消重复请求
  if (pendingRequests.has(cacheKey)) {
    pendingRequests.get(cacheKey)?.abort();
  }
  
  const controller = new AbortController();
  config.signal = controller.signal;
  pendingRequests.set(cacheKey, controller);
  
  // 检查缓存（仅 GET 请求）
  if (config.method === 'get') {
    const cached = cache.get(cacheKey);
    if (cached && Date.now() - cached.timestamp < CACHE_DURATION) {
      return Promise.reject({ __CACHE__: cached.data });
    }
  }
  
  return config;
});

instance.interceptors.response.use(response => {
  const cacheKey = generateCacheKey(response.config);
  pendingRequests.delete(cacheKey);
  
  // 缓存 GET 请求结果
  if (response.config.method === 'get') {
    cache.set(cacheKey, { data: response.data, timestamp: Date.now() });
  }
  
  return response;
});
```

**原因**：缓存减少重复请求；取消重复请求避免资源浪费和数据竞态。
